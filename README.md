# Insurance Partners MVC

## Features

* Display a list of all partners
* Search partners from the partner list
* Show partner details in a modal window
* Show partner policies inside the partner details modal
* Adding a new partner
* Adding a policy for an existing partner
* Highlight newly added partner after successful save
* Mark partners with `*` when they have more than 5 policies or total policy amount greater than 5000
* Store partner and policy data in a PostgreSQL database
